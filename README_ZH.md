# 未知回响

> 仍在开发中，欢迎反馈问题，当前内容以第一维度"回声境域"为主,更多维度与能力规划在后续版本，预计将有3个维度，6个主线BOSS，5种能力/神器。

> 走出去,聆听大地的回响,发现被遗忘的世界。

**未知回响**是一个以"探索 + 解谜 + Boss 战 + 渐进式能力成长"为核心的冒险类 Mod。

你将打开通往**回声境域**的大门,在十个风格迥异的群系里追踪远古回声、破解古老机关、击败守护者,逐步习得三种"回响"之力,并最终在悬崖大档案馆完成属于你个人的"毕业"。

---

## 快速开始

1. **合成回声境域钥匙**:用**回响碎片 + 回响核心**合成一把*回声境域钥匙*。
2. **找到残响信标**:残响信标会在主世界自然生成(残响信标遗迹)。
3. **唤醒信标**:手持钥匙右键信标。
4. **进入回声境域**:右键已激活的信标。
5. **返回主世界**:在境域内右键信标即可回到主世界出生点/重生点。

---

## 核心玩法

1. 探索**回响森林**,找到记忆柱庭院,击败**遗忘巨像**,解锁**风之回响**。
2. 探索**镜湖 / 潮汐灯塔礁**,深入沉没圣殿,击败**深渊观测者**,解锁**潮汐回响**。
3. 探索**镜尘回廊**,攻入镜面神殿,击败**镜像守护者**,解锁**真视回响**。
4. 在 10 个群系收集材料、远古残页、结构发现,挑战 8 个**区域守护者(Mini Boss)**。
5. 凭探索进度解锁并升级 3 件**核心神器**,强化三条能力。
6. 找到**回声峭壁**的大档案馆,达成完成度后领取个人毕业奖励与通往更深处的线索。

## Boss 与区域守护者

### 三大主线 Boss

每个主线 Boss 都有**破防机制**:未破防时硬抗高伤害,必须先用场地机关打开破防窗口才能造成有效伤害。

| 主线 Boss | 场地 | 解锁能力 |
|---|---|---|
| **遗忘巨像** | 记忆柱庭院(回响森林) | 风之回响 |
| **深渊观测者** | 沉没圣殿(镜湖/潮汐灯塔礁) | 潮汐回响 |
| **镜像守护者** | 镜面神殿(镜尘回廊) | 真视回响 |
